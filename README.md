# Розпізнавання номерних знаків та ідентифікація типів транспортних засобів з використанням deep learning

В цьому репозиторії знаходиться код та результати Випускної Кваліфікаційної Роботи із створення моделі розпізнання українських номерних знаків з використанням YOLO (You Only Look Once). Використовувалась власна вибірка зображень та анотація. Будуть опубліковані окремо в майбутньмоу.

Використанні технології:

- Python 12.6
- Ultralitics YOLO 11
- EasyOCR
- [galfar/deskew](https://github.com/galfar/deskew) для вирівнювання тексту

## Вибірка даних

Для тренування та оцінювання моделі було створено власний датасет, що перебуває у папці `dataset/`.
В цій папці можна знайти поділений датасет на test, train, val. Також там знаходяться мітки типу транспортного засобу та значення його номерного знаку.
